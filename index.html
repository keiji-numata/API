API Description

I selected a free sports api that allows me to get data about sports teams, players, and events. Although there was a lot more information that was provided, I chose to exclusively display the team description to save space.

Website

https://www.thesportsdb.com/api.php

Parameters

I needed to specify that I was conducting a search for teams, and then also the team name

Applications

This API has multiple applications, especially for sports fans, serving as a sort of encyclopedia for sports information. One example would be when searching for a certain sports players statistics, and another example would be researching about a specific sports team.
